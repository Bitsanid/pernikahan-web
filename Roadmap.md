# Roadmap - Wedding Invite SaaS

## Versi SaaS MVP (v2.0) - Q1 2026 (Sekarang)
- Migrate dari MVP: Tambah auth/backend.
- Library: 10+ templates wedding themes.
- Editor custom + generate PDF.
- Integrasi Stripe payment.
- Deploy live dengan domain custom (Namecheap gratis).
- AI tasks: Generate 5 new themes via Claude.

**Timeline**: 2-4 minggu. Milestone: First paying customer.

## Versi 2.1 - Q2 2026
- Bulk generate/send via WhatsApp (Bannerbear + Zapier).
- Admin dashboard.
- Mobile app PWA.

**Timeline**: 1 bulan.

## Versi 3.0 - Q3 2026
- Expand themes: 50+ (user-upload).
- Analytics/revenue tracking.
- Integrasi Canva/Templett untuk import templates.
- Monetisasi lanjutan: Affiliate program.

**Timeline**: 2 bulan.

Long-term: Scale ke event lain (birthday, etc.), target 1K users. Review bulanan!
